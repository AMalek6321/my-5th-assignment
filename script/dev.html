Question-1: What are the different ways to select an element in the DOM?

Answer: There are three main ways to select an element in the DOM:
1. By ID
2. By class
3. By tag name

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML returns the HTML code of the element, innerText returns the text content of the element, and textContent returns the text content of the element without any HTML tags.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a useful technique that makes event handling in JavaScript simple. This sets a single event handler to their common parent element, rather than using separate event handlers for multiple child elements.

Question-4: What is event bubbling in the DOM?

Answer: When a single event is triggered or executed multiple times it is called event bubbling

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:1. Creating Elements:This method is used to create a new HTML element.
document.createElement()
2. Adding Elements:This method is used to add an element as a child of another element.
parentNode.appendChild(childNode)
3. Removing Elements:This method is used to remove an element from the DOM.
parentNode.removeChild(childNode)
